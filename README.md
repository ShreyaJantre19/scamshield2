# ScamShield

A project for scam detection and prevention.

## Project Structure

This project is divided into three main components:
- **Frontend**: Client-facing application.
- **Backend**: Service APIs and database management.
- **AI Integration**: AI models and logic for scam classification.

## Branches

- `main`: Production-ready code.
- `frontend`: Frontend development branch.
- `backend`: Backend development branch.
- `ai`: AI integration and machine learning models branch.
