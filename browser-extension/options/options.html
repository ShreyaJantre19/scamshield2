ScamShield
Extension Settings
API Configuration
Default: http://localhost:8000 (local backend). Change this to your deployed backend URL if hosted remotely.
How to use
🔗 URL Check: Click the extension icon → URL tab → paste link → Analyze.

📧 Email Check: Email tab → paste sender, subject, body → Analyze.

🖱️ Right-click: Right-click any link → "🛡️ Check with ScamShield".

📄 Page Scan: Page tab → Scan all links → badges appear next to risky links.

💬 WhatsApp: Forward suspicious messages to your ScamShield WhatsApp bot number.
